## 🔄 更新徽章

徽章会自动更新，但您也可以手动更新：

1. 访问 [shields.io](https://shields.io/)
2. 选择徽章类型
3. 输入相关信息
4. 复制生成的URL
5. 更新文档中的徽章链接
